# Smile Moore Website Structure Plan

## Overview
Create a professional dental practice website with a compelling landing page and comprehensive information sections.

---

## Site Structure

```
smilemoore.co.uk/              → Landing Page (Lead Capture)
smilemoore.co.uk/home          → Main Website/Practice Info
smilemoore.co.uk/services      → Services Detail Page
smilemoore.co.uk/about         → About Us & Team
smilemoore.co.uk/contact       → Contact & Booking
smilemoore.co.uk/book          → Online Booking Form
smilemoore.co.uk/prices        → Treatment Prices

Campaign Pages (Tracking):
smilemoore.co.uk/earlybird     → Early Bird Campaign
smilemoore.co.uk/leaflet       → Leaflet Campaign
smilemoore.co.uk/poster        → Poster Campaign
smilemoore.co.uk/card          → Business Card Campaign
```

## Page Breakdown

### 1. **Landing Page** (`smilemoore.co.uk/`)

**Purpose:** Convert visitors immediately - capture leads before they leave

**Content:**
- **Hero Section:**
  - Bold headline: "Get Your Perfect Smile Today"
  - Subheading: "Limited Time: Early Bird Special for New Patients"
  - Large prominent CTA button: "Claim Your Offer"
  - Background: Professional dental image or gradient

- **Offer Section:**
  - Clear value proposition
  - Special offer details (e.g., "£50 off first treatment" or "Free consultation")
  - Urgency element: "First 50 patients only"
  - Trust signals: "No hidden fees" • "Flexible payment plans" • "Same-day appointments"

- **Booking Form (Above the Fold):**
  - Name
  - Phone
  - Email
  - Preferred time
  - "Book Free Consultation" button

- **Quick Benefits (Icons + Text):**
  - ✓ Experienced dentists
  - ✓ Latest technology
  - ✓ Flexible appointments
  - ✓ Payment plans available

- **Social Proof:**
  - Patient testimonials (when available)
  - "Join 500+ happy patients" (update with real numbers)
  - Trust badges: GDC registered, Awards, etc.

- **Final CTA:**
  - "Ready to transform your smile?"
  - Phone number (large, clickable)
  - Book button
  - Link to full website: "Learn more about our practice →"

**Design:**
- Clean, modern, mobile-first
- Blue/white color scheme (professional, trustworthy)
- Minimal navigation (keep focus on conversion)
- Fast loading
- Single page scroll

## User Journeys

### Journey 1: QR Code Scanner (High Intent)
```
Scan QR → Landing Page → Fills form → Confirmation
                      ↓
                   Calls practice directly
```

### Journey 2: Researcher (Low Intent)
```
Landing Page → "Learn More" → Home Page → Services → About → Prices → Contact/Book
```

### Journey 3: Direct Visit
```
smilemoore.co.uk → Landing Page → CTA
                 ↓
              /home → Browse → Book
```

### Journey 4: Campaign Tracking
```
QR (leaflet) → smilemoore.co.uk/leaflet (landing) → Form/Call
QR (poster) → smilemoore.co.uk/poster (landing) → Form/Call
```

## Design Guidelines

### Colors:
- Primary: Blue (#2563eb) - Trust, professionalism
- Secondary: White (#ffffff) - Clean, medical
- Accent: Light blue (#dbeafe) - Highlights
- Text: Dark gray (#1f2937) - Readable
- Success: Green - CTAs, confirmations

### Typography:
- Headings: Bold, modern sans-serif
- Body: Clean, readable sans-serif
- Size: Minimum 16px body text

### Imagery:
- Professional dental photos
- Friendly team photos
- Modern, clean clinic photos
- Diverse patient representation
- High quality, compressed for web

### Spacing:
- Generous white space
- Clear visual hierarchy
- Easy to scan
- Mobile-first approach

## Conversion Optimization
